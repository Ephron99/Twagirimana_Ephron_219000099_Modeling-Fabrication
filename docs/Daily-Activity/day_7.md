# 7. Activity of Day 7

## Digital Fabrication III: CNC Router Milling & Cutting

#### Overview

A CNC removes material from a workpiece based on tool paths generated from digital designs. This method is commonly used to manufacture parts from materials.

The machine follows instructions generated from software and moves along multiple axes to cut the material accurately.

**Before machining begins** I Secure the material on the machine bed, Install the appropriate cutting tool, Set the machine to zero reference point (origin) then Load the machining program. Once the setup is complete, the CNC router begins cutting the material according to the defined toolpaths.

!!! Safety
Working with CNC requires proper safety practices. Keeping hands away from moving tools, and Monitoring the machine during operation.
